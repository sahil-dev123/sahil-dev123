# Featured Engineering Projects

## HybridAI — Intelligent Hybrid Retrieval Infrastructure

> Production-grade AI retrieval platform combining:

- Knowledge Graphs
- RAG Pipelines
- PostgreSQL Full Text Search
- Vector Similarity Search
- Intelligent Query Routing

### Key Engineering Features

- Hybrid semantic + exact retrieval
- Query orchestration engine
- Graph-based contextual traversal
- PostgreSQL TSVECTOR + GIN indexing
- Reduced hallucinations via layered retrieval
- Optimized retrieval latency
- Async ingestion pipelines

### Architecture Highlights

```text
User Query
    ↓
Query Router Engine
    ↓
 ┌───────────────┬───────────────┬───────────────┐
 │               │               │
Graph Search   Vector Search   PostgreSQL FTS
 │               │               │
 └───────────────┴───────────────┘
                ↓
         Context Aggregation
                ↓
               LLM
                ↓
            Final Response
```

### Engineering Decisions

- Used PostgreSQL FTS for deterministic exact retrieval
- Used Vector Search for semantic understanding
- Used Query Routing to reduce unnecessary LLM calls
- Combined graph traversal for relationship-aware retrieval
